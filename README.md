This project is for testing and demoing ScalaPB over ScalaJS.

Live demo
---------

Go to http://thesamet.github.io/scalapbjs-test/

Run the utests
--------------

In SBT, run `test`.

Try in a browser
----------------

In SBT, run `fastOptJS`

Then open index-dev.html (from this directory) in a browser, and watch the
console.
